# Beam Remote
## About
Beam remote is a web-based tool which allows touch screen devices to be used as a customisable remote for BeamNG.drive.<br>
Note: This tool is not at all affiliated with or endorsed by the developers of BeamNG.drive, it is simply a community made tool

# Please note this tool only currently works on Linux (Does not work on Windows!)
