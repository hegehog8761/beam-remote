# Installation
1. Download all of the files (by pressing the Code button on the main page then download as zip and extracting the ZIP file)
2. Open the folder containing the file named "main.py" in your terminal
3. Use python to run the main python file (`python3 main.py`)
4. Connect to the address given when the address (the part after Running on) which starts with 192.168 on your chosen device
